# H5 Game Collection

这是一个精心整理的H5小游戏集合项目，支持一键部署到个人域名站点。通过简单的配置，您可以在自己的网站上轻松运行多款有趣的H5游戏。

## 项目特点

- 🎮 精选优质H5小游戏
- 🚀 一键部署功能
- 🔗 支持自定义域名
- 📱 移动端友好
- 🛠️ 简单易用的部署流程

## 游戏列表

每个游戏项目包含以下信息：
- 一键部署按钮
